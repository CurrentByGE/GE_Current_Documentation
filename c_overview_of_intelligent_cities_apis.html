Intelligent Cities API Overview

Each Intelligent Cities API provides asset and location information, with additional URI resources and data collected from sensors installed on intelligent lighting fixtures in roadways and parking areas. Each API service is described below.

Note: Before using these APIs, start with the /assets or /locations endpoints described in the General APIs section.

Parking Planning

The Parking Planning service provides parking metadata that is collected from intelligent lighting sensors installed along public roadways or in parking lots. Each time a vehicle enters or leaves a parking space, a timestamped event is created and stored in the Predix cloud.

This data can be obtained from a single sensor or a group of sensors, as needed, and can provide valuable insights about the parking activity in a given region. The data can also be used to improve safety and parking convenience.

See the Vehicle In, Vehicle Out API.

Pedestrian Planning

The Pedestrian Planning service provides pedestrian metadata that is collected from intelligent lighting sensors installed along public roadways or in parking lots.

As pedestrians enter or leave a region of interest, the service creates and stores a timestamped event, counting each event. This data can be obtained from a single sensor or a group of sensors, as needed, to provide insights to city officials and local businesses about pedestrian usage in a given region. The data can also be used to enhance safety and convenience.

See the Get Pedestrian Data API.

Traffic Planning

The Traffic Planning service provides vehicle metadata collected from intelligent lighting sensors installed along public roadways.

As vehicles pass by these sensors, information such as speed, direction, lane use, and volume (counts) are collected and stored with a timestamp in the Predix cloud. The data can be viewed from any number of intelligent street lights in a given region and can present both historical and near real-time information.

The density, range, and flexibility of this sensor data can provide insights into traffic patterns at different intervals. Along with facilitating the planning for future safety and convenience enhancements, the data can reveal opportunities to optimize traffic flows.

One response is provided for each car that traverses the area monitored by the device.

See the Get Traffic Flow API.

Public Safety

The Public Safety service provides access to images, video, and other media collected from intelligent lighting sensors installed along public roadways or parking lots.

The media can be requested using a date range starting near real-time and ending based on availability. This data can be obtained from a single sensor or a group of sensors, as needed, to provide insights to city officials that enhance awareness. This service can be used in concert with other Intelligent Environments services to enhance a use case of interest.

Images are captured for a variety of scenarios for vehicles and pedestrians.

See the Get Media API.
