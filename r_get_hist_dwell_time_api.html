Get Historical Dwell Time API

The Historical Dwell Time API measures the total amount of time per the device for the “session of dwelling” within a department or zone.

Dwell times are calculated based on the entrance and exit of departments or zones, enabling the triggering of actions.

METHOD: GET
URI: <production url>/v1/locations
SAMPLE URI: <production url>/v1/locations/1000000092

Query Parameters

Parameter Description Required? Values
id GE identifier for a location Yes 1000000092
analytic-name
  • For Dwell Time by Department, the dwell zone.
  • For Dwell Time by Device, the dwell device.
 Yes
  • For Dwell Time by Department, a String, such as DWLZONE.
  • For Dwell Time by Device, a String, such as DWLDVCE.
analytic-category For both Dwell Time by Department and Dwell Time by Device, the dwell time. Yes. A String, such as DWELL_TIME.
start-ts Start time Yes Number, with a value such as 1459148400000.
end-ts End time Yes Number, with a value such as 1459212163672.

Response Parameters

Parameter Data Type Required? Description
analyticName String Yes Either BY-DEPARTMENT or BY-DEVICE
properties String Yes Identifies the dwell time either by department or by device:
  • For DEPARTMENT, the department or zone and time spent in that zone are given, expressed as a string. For example, "A": 29 specifies that 29 seconds were spent in department A.
  • For DEVICE, the following information is provided:
    • DEVICE-ID: Unique identifier established by a customer or external resource for the asset monitoring the location. For example, "device_1".
    • SESSION-ID: ID of the specific session, expressed as an alphanumeric string such as 6403d8b8-43cf-40a1-bdb9-1cd876d96de7.
    • SESSION-START: Time the session started, expressed in seconds as a LONG number, such as 1457935539745.
    • SESSION-END: Time the session ended, expressed in seconds as a LONG number, such as 1457935730102.
    • DWELLTIME-BY-DEPARTMENT: Time spent in a particular department or zone. For example, "A": 31, specifies that 31 seconds were spent in department A.
